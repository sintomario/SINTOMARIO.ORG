# SINTOMARIO.ORG

Repositorio de publicacion para `sintomario.org`.

La generacion del sitio se ejecuta 100% en local. Este repo debe contener solo artefactos de deploy:

- HTML generado
- `CNAME`
- `robots.txt`
- `sitemap.xml`
- `404.html`
- assets publicos estrictamente necesarios

El motor, el corpus, los scripts y cualquier credencial viven fuera del repo publico.
