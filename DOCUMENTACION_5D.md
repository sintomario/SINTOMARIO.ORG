## 🚀 Sistema de Índices 5D (5X XXXXX)

### Arquitectura del Sistema

El sistema de índices 5D implementa una indexación multidimensional capaz de gestionar **10,000+ artículos** con diversidad e impredecibilidad controladas.

#### Dimensiones del Índice
- **Territorio** (0-4): cuerpo, mente, espiritu, social, ambiental
- **Sistema** (0-9): organicos, energeticos, emocionales, etc.
- **Profundidad** (0-9): superficial a profundo
- **Frecuencia** (0-9): raro a comun
- **Complejidad** (0-9): simple a complejo

#### Formato del Índice
```
SINTO-XXXXX (5 dígitos)
Ejemplo: SINTO-30675
```

#### Matriz AMS-Risomática
- **AMS**: Asociación Multidimensional Semántica
- **Risoma**: Red de Índices Semánticos Orgánicos Matemáticos
- **81 códigos posibles** (9x9 combinaciones)
- Códigos como: AMS-06, RIS-02, OMA-07, SOM-02, ARI-08

#### Entropía Controlada
Cada índice calcula entropía de Shannon para garantizar diversidad:
- Rango: 0.0 - 3.0 bits
- Promedio: ~2.1 bits
- Impide patrones predecibles
